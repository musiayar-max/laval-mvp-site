# LAVAL Legacy Configurator Asset Reference

This folder is for developer reference only. It is not part of the active MVP configurator.

Current MVP direction:
- Full intact mantel preview images.
- Enabled controls: Family, Profile, Finish.
- Disabled controls: Legs, Shelf / Crown, Relief, Door / Firebox.
- No live component layer stacking.
- No CSS offsets to rescue bad layer assets.

The old generated/stage assets were archived because they are no longer used by the live configurator. They remain useful as visual references, naming references, and historical logic references for a future developer.

Active configurator assets now live under:

```text
configurator/assets/preview-families/
```

Disabled-control UI icons now live under:

```text
configurator/assets/disabled-icons/
```

Do not reconnect the old generated/stage folders to the live configurator without a deliberate rebuild.
