# soft-sculpted preview family pack v0.3.0

This is an MVP preview-selector pack. It uses full pre-rendered mantel images for each finish selection, not component stacking and not texture overlays.

Installed live paths:
- images/: full 2400 x 1800 transparent PNG preview images
- icons/: 1024 x 1024 selector icons
- source/: exact user-provided source images retained for developer audit
- qa/: working cutout artifacts, not referenced by live UI

Live UI must reference only images/ and icons/.
